
## Installation Project

- Clone this project 

```bash
  git clone https://github.com/fahmiarif/test-fe-scp.git
```

- Enter to project directory

```bash
  cd test-fe-scp
```

- Install dependencies
```bash
  npm install
```
- Run project
```bash
  npm run dev
```
- Open browser and then enter url
```bash
  http://localhost:3000
```
